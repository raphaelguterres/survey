# Survey — Topografia, Hidrometria & Hidrologia

Landing page da **Survey** — topografia, georreferenciamento, hidrometria e hidrologia.

Página estática de arquivo único (`index.html`), sem build. Three.js e GSAP são carregados via CDN.

## Ver localmente
Basta abrir `index.html` no navegador, ou servir a pasta:

```bash
python -m http.server 8000
# acesse http://localhost:8000
```
